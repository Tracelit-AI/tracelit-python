You are integrating the Tracelit Python SDK.

1. Install:
   pip install tracelit

2. Initialize early in process startup:
   import tracelit
   tracelit.auto_start(
       api_key=os.environ["TRACELIT_API_KEY"],
       service_name="MY_SERVICE",
       environment=os.getenv("ENV", "production"),
       sample_rate=1.0,
   )

3. Disable in tests:
   TRACELIT_ENABLED=false

4. Optional middleware:
   from tracelit.integrations.asgi import TracelitASGIMiddleware
   app = TracelitASGIMiddleware(app)

5. Manual spans and metrics:
   with tracelit.tracer().start_as_current_span("op"):
       pass
   tracelit.metrics.counter("my.events").add(1)

What you get automatically:
- Traces for supported frameworks and libraries.
- Logs bridged into OpenTelemetry log pipeline.
- Runtime/process metrics with periodic export.

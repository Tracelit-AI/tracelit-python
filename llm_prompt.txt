You are integrating the Tracelit Python SDK.

1. Install:
   pip install tracelit

2. Install OpenTelemetry instrumentation packages for libraries in use
   (django, flask, fastapi, redis, celery, psycopg2, pymysql, requests, etc.).
   Example:
   pip install opentelemetry-instrumentation-django opentelemetry-instrumentation-redis

3. There is NO tracelit.init() API. Use tracelit.auto_start().

4. Django (recommended):
   - Add to INSTALLED_APPS (first):
     "tracelit.integrations.django.TracelitConfig"
   - Do NOT call tracelit.auto_start() at the top of settings.py.
   - Set TRACELIT_* environment variables (or bridge decouple/django-environ into os.environ).
   - Celery workers: install_celery_hook() in celery.py.

5. Flask / WSGI:
   - tracelit.auto_start(...) before app handles requests
   - optional: app.wsgi_app = TracelitWSGIMiddleware(app.wsgi_app)
   - pip install opentelemetry-instrumentation-flask

6. FastAPI / ASGI:
   - tracelit.auto_start(...) before app handles requests
   - optional: app = TracelitASGIMiddleware(app)
   - pip install opentelemetry-instrumentation-fastapi

7. Generic scripts:
   import tracelit
   tracelit.auto_start(api_key=..., service_name=..., environment=...)

8. Disable in tests:
   TRACELIT_ENABLED=false

9. Manual spans and metrics:
   with tracelit.tracer().start_as_current_span("op"):
       pass
   tracelit.metrics.counter("my.events").add(1)

What you get automatically (when instrumentation packages are installed):
- Traces for supported frameworks and libraries.
- Logs bridged into OpenTelemetry log pipeline.
- Runtime/process metrics with periodic export.
